🔗 產業關係圖產生器

📋 精簡格式（推薦）

AI 只需輸出以下結構，關係直接寫在每個公司身上，不用分開寫 links。

最小結構

{
  "title": "圖表標題",
  "cores": [
    ["核心公司A", "描述"],
    ["核心公司B", "描述"]
  ],
  "coreLinks": [
    ["公司A", "公司B", "上下游"]
  ],
  "nodes": [
    ["客戶X", "customer", "描述", [
      ["核心公司A", "供應商"],
      ["核心公司B", "材料供應"]
    ]]
  ]
}

cores 陣列

核心公司：["名稱", "描述"]

自動設為 type: "core"

coreLinks 陣列（選填）

核心之間的關係：["A", "B", "標籤"]

nodes 陣列

每個元素：["名稱", "type", "描述", [[連線]...]]

• 描述可省略：["名稱", "type", [[連線]...]]

• 連線也可省略：["名稱", "type", "描述"]

• 每條連線：["對象名稱", "關係標籤"]

type 類型

core 核心主體（只在 cores 中定義）
customer 客戶
competitor 競爭對手
supply 供應鏈
subsidiary 子公司
driver 平台方

自動化功能

✦ 節點大小：依連線數自動計算

✦ 節點位置：依連接的 core 自動分群

✦ 連線顏色：依所屬 core 自動著色

✦ 核心佈局：自動三角/圓形排列

✦ 圖例：自動根據資料生成

✦ 兩種格式都支援（向下相容完整格式）
